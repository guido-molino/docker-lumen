# Docker - Lumen - MySQL
> A dockerized lumen environment designated for local development

## Installation

1. `git clone`
2. `cp .env.example .env`
3. `docker-compose up`

### Others..

You can run the following commands in order to access to the container's shell

1. `docker ps`
2. `docker exec -it <lumen-app-id> /bin/sh `
